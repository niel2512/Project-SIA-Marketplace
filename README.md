## Installation

Clone the repo locally:

```sh
git clone https://github.com/niel2512/Project-SIA-Marketplace.git
```

```sh
composer install
```

Setup configuration:

```sh
cp .env.example .env
```

Edit .env `DB_DATABASE = "name database"`

Generate application key:

```sh
php artisan key:generate
```

Run database migrations:

```sh
php artisan migrate
```
